# 🏥 Hospital Management System

## 📖 Overview
Hospital Management System is a web-based application developed using Java Servlets, JSP, JDBC, and MySQL. The system helps manage patient records, doctor information, appointment scheduling, and employee authentication through a centralized platform.

## 🚀 Features
- Employee Login & Session Management
- Add New Patients
- View Patient Records
- Update Patient Information
- Delete Patient Records
- View Doctor Details
- Book Appointments
- Doctor Availability Validation
- MySQL Database Integration

## 🛠️ Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat 9
- Maven

## ✨ Key Highlights
- Developed a web-based Hospital Management System to manage patient registration, doctor records, and appointment scheduling.
- Implemented full CRUD operations for patient data using JDBC and MySQL.
- Built a secure employee login system with session-based authentication.
- Designed dynamic JSP pages with a user-friendly interface for hospital staff.

## 📂 Modules

## ⚙️ Database
The application uses MySQL to store:
- Employee Details
- Patient Records
- Doctor Information
- Appointment Details

## ▶️ How to Run
1. Clone the repository.
2. Import as a Maven Project.
3. Configure MySQL database credentials.
4. Start Apache Tomcat 9.
5. Run the application.
6. Open the project in your browser.

## 🔐 Authentication
Only authorized employees can access the system through a secure login mechanism with session management.
